## Decisions Made
- **Decision**: Stop hook exits 0 (advisory) not 2 (block) — **Rationale**: Stop fires on every turn; exit 2 causes infinite loop
- **Decision**: `.ai-context-setup/` as new home for SETUP-PROMPTS — **Rationale**: ships to target project without source repo; avoids agent confusion from flat `.ai-context/`
- **Decision**: Skip hooks merge when existing `"hooks"` key present — **Rationale**: bash string merge can't safely merge JSON with duplicate keys; conservative skip with manual instruction is safer
- **Decision**: Repo slug stays `ai-context`; display name becomes "AI Context: Shared Memory for Coding Agents" — **Rationale**: slug matches `.ai-context/` convention; branding updated without breaking URLs
